Some terminology

Object

An object is simply a container for variables and methods. Every class inherits from the Object type (more on inheritance later).

Class

A class is a type of object that Java recognizes when compiling and executing code, not at the object level. Almost every bit of code in Java is within a class. Similar to C using function calls, Java uses classes as the simplest level of abstraction.

Method

A method is a function within a class. By function I mean that there is code to be executed when called. The method is found by referencing the method name within the Object's scope. In Exercise 1 below, println() is a method of the System Object.

Instance variable

Also called fields or properties in other languages, they comprise variables that are used within the scope of the class. When writing a Person class, a String field that contains the name of the Person being stored would be useful.

Install the JDK

To just execute Java you need the Runtime Environment which is on the majority of machines. We want to write and execute our Java code. Luckily Java is free and the JDK (Java Development Kit) is readily available.

Java 8 JDK link

When downloading make sure to get the correct verision (32 bit or 64 bit) for your particular machine. To check bring up the command line for your machine.

For windows users, navigate to the control panel. In the control panel, go to the System icon. Within System, under the general tab, will display the operating system information including the bit type.

For Mac users, open the Terminal. The Terminal program default location is within Finder > Applications > Utilities > Terminal. Type "uname -m" without quotes. x86_64 is 64 bit, i686 is 32 bit.

Linux based users, open the command line. Type "uname -m" without quotes. x86_64 is 64 bit, i686 is 32 bit.

Once the download is complete, open the file to run the executable which will install the SDK on your machine.

If not already open, bring up the command line.

There will be some reliance on the command line over using the UI based file systems (such as Windows Explorer). So it best to start learning now.

Bringing up the command line

For windows, search for "cmd" in the Start Menu.

For Mac, see above in the SDK installation.

For Linux, you probably know where the command line is already.

Once open, navigate to a desired work directory. This is done using the 'cd' command which stands for Change Directory. Directory is another name for a folder. This command works on Windows, Mac, and Linux.

When at the desired location, create a directory with 'mkdir' for Linux/Mac users or 'md' for Windows.

Compiling and running the code

At the command line, navigate to the Lesson 1 folder. Type 'javac Hello' and press tab to auto-complete the HelloWorld.java file. This will compile the code into HelloWorld.class file.

Run the code with 'java Hello' and again tab to complete the proper sequence. Note you do not need to add 'HelloWorld.class' as the computer will know that 'HelloWorld' is executable.

Note at the console in the next line, "Hello World!" should be printed as the program ran successfully or a compilation error happened.
